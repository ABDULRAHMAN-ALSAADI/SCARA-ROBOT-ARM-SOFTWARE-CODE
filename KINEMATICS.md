# Kinematics

## Forward Kinematics

Forward Kinematics calculates the end-effector position from joint angles.

Inputs:

```text
θ1
θ2
```

Outputs:

```text
X
Y
```

---

## Inverse Kinematics

Inverse Kinematics calculates joint angles from a desired position.

Inputs:

```text
X
Y
```

Outputs:

```text
θ1
θ2
```

---

## Workspace

The reachable workspace depends on:

```text
L1
L2
```

Changing these values requires recalibration.
