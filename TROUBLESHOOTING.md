# Troubleshooting

## Robot Moves In Wrong Direction

Check:

- Motor wiring
- Direction settings

---

## Homing Fails

Check:

- Limit switch wiring
- Switch polarity
- Homing direction

---

## Positioning Error

Check:

- Link lengths
- Microstepping settings
- Steps per revolution

---

## Web Interface Not Accessible

Check:

- WiFi configuration
- ESP32 IP address
- Browser console logs

---

## Recipe Execution Stops

Check:

- Stored recipe validity
- Motion limits
- Error logs
